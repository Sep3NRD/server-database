<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/Sep3NRD/server-database
```

<p>2. Navigate to the directory where the pom file is</p>

<p>3. Maven clean install</p>

```
mvn clean install
```
